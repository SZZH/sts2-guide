# StS2 内容执行规则（2026-04-01）

## 适用范围

本规则仅适用于当前确认可执行的 3 个主题：

1. `Exhaust`
2. `Necrobinder Common Cards`
3. `Celestial Might`

不适用于待核实实体词、版本真值未确认内容、以及需要新增官方映射的数据页。

## 全局规则

- 只写已确认事实、玩家常见观点、或编辑部明确标注为争议点的内容
- 不写未核实数值、概率、强度排序、覆盖范围
- 不把高热度玩家说法写成官方口径
- 不写“版本答案”“最强”“核心”“必带”“稳定收益”这类强结论
- 不为了内链新增没有真实承接价值的聚合页

## 句型权限

所有句子只能属于以下 3 类之一：

1. 可直接确认的事实
2. 玩家讨论中的常见观点
3. 编辑部不下结论、仅说明争议

不属于这 3 类的句子，不进入正文。

## 主题词表

### Exhaust

- 主词：
  - `Exhaust`
  - `exhaust mechanic`
  - `sts2 exhaust`
- 允许说法：
  - `Exhaust build`
  - `remove exhaust`（仅用于解释搜索意图，不可暗示游戏内存在该系统机制）
  - `deck thinning`
- 禁用说法：
  - `best exhaust build`
  - `meta exhaust build`
  - `current strongest exhaust cards`

### Necrobinder Common Cards

- 主词：
  - `Necrobinder common cards`
  - `Necrobinder card pool`
- 允许说法：
  - `early stability`
  - `Doom line`
  - `Osty line`
  - `Soul support`
- 禁用说法：
  - `best Necrobinder commons`
  - `must-pick common cards`
  - `tier list`
  - `complete best list`

### Celestial Might

- 主词：
  - `Celestial Might`
  - `celestial might guide`
  - `celestial might sts2`
- 允许说法：
  - `payoff card`
  - `bait`
  - `Stars shell`
  - `Forge shell`
- 禁用说法：
  - `buffed in the current patch`
  - `nerfed in the current patch`
  - `best Regent payoff`
  - `must-take`

## 每篇文章只回答什么

### Exhaust

- 只解释机制、节奏作用、误区、以及搜索词 `remove exhaust` 的真实意图
- 不做版本真值判断

### Necrobinder Common Cards

- 只做 common cards 的阅读框架、分类理解、以及早期稳定性认知
- 不做卡表排行和强度定论

### Celestial Might

- 只做单卡定位说明、争议点整理、何时值得拿/跳过的条件判断
- 不做未确认升级态和数值判断

## 参考来源规则

### 事实层

仅允许使用：

- 官方公告 / 官方 patch notes
- 当前已确认的站内结构化数据
- 游戏内可复现文本

### 玩家观点层

仅允许使用：

- Reddit / Steam 等高互动讨论
- 这些来源只能用于：
  - 误区
  - 打法理解
  - 条件判断
  - 玩家争议点

禁止用玩家来源直接证明：

- 数值真值
- 版本强度
- 最优构筑
- 官方设计意图

## 内链规则

- 每个入口页最多给一个主链接
- 每篇正文最多链出 2 到 3 个页面
- 三篇主落地页彼此最多只允许 1 条自然互链
- 锚文本只用直接意图词，不用夸张词

## 停止条件

执行中一旦出现下面任一情况，立即停止：

- 需要引用未确认 patch 数值
- 需要把玩家经验写成定论
- 发现目标词会和现有页面抢词
- 发现某个结论必须依赖待核实实体页
